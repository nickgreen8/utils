readFile()

readFile(string  $file, boolean  $array = false) : mixed

This function is used to read the conetent of a JSON file. The path and file name is passed first and if the content is required in array format, a boolean true should also be passed. If there is an error, NULL is returned. If there is no error, the JSON is returned as an object or an array.

Parameters

string $file

File path and name
boolean $array

Incicator for an array to be returned (Default: false)

writeToFile()

writeToFile(mixed  $data, string  $file) : boolean|void

This function is used to write out a JSON file. The data that is to be written must be passed as an array, string or an object. The file path and name must then also be passed.

If there is an error NULL is returned. If there is no error, true is returned.

Parameters

mixed $data

The data to be encoded and then written
string $file

The path to the file

Returns

boolean|void
